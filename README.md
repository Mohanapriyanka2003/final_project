# AI-Enabled Underwater Object Detection System

An ESP32-based underwater sensing and monitoring system designed for sonar mapping, object tracking, and machine learning-based object classification using ultrasonic sensor data.

## Project Overview

This project focuses on detecting and classifying underwater objects using AI and embedded systems technologies. The system integrates ultrasonic sensing, ESP32 microcontroller processing, and machine learning algorithms to improve underwater object identification and monitoring.

## Features

- Underwater object detection
- Sonar-based distance mapping
- ESP32 integration
- Real-time object tracking
- Machine learning-based classification
- Data preprocessing and analysis

## Technologies Used

- ESP32
- Python
- Machine Learning
- Ultrasonic Sensors
- Embedded Systems
